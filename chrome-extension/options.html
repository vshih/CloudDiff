External Diff Tool

  1. Install the CloudDiff Helper:
    • Download and open clouddiff-helper_windows.exe.
    • To bypass Windows SmartScreen, click the More info link, then the Run anyway button.
    • Open a Terminal window.
    • Run the following command: curl -L https://bitbucket.org/vshih/clouddiff-helper/downloads/install.sh | sh
  2. Install a diff tool (kdiff3, for example).
  3. Specify the command line to run your diff tool, using the full path if required. Quote the path if it contains spaces.
  4. Optional: use $1 and $2 to represent the left and right selected files passed.
    • If not given, the file names will be quoted and appended as the first and second parameters to your program.
